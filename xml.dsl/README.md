xml.dsl
=======
The xml.dsl library contains a DSL builder for XML and DSLs for common some XML Dialects.
Included XML DSLs:
* BPMN
* Docbook
* XSL-FO
* MathML
* RDF
* SVG
* XHTML
* XSD
* XSL
